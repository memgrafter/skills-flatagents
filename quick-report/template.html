Methodology

The model uses ordinary least squares regression. The cost function is:

$$J(\theta) = \frac{1}{2m} \sum_{i=1}^{m} \left( h_\theta(x^{(i)}) - y^{(i)} \right)^2$$

Where \(m\) is the number of training examples, \(h_\theta(x)\) is the hypothesis function, and the gradient is computed as \(\nabla_\theta J = \frac{1}{m} X^T (X\theta - y)\).
