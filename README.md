# Claude Skills - FlatAgents

Claude Code skills powered by [FlatAgents](https://github.com/anthropics/flatagents).

## Skills

- **search-refiner** - Search the web and refine results to key findings using Cerebras + Exa

## Install

```bash
cd search_refiner
./install.sh
```

## Requirements

- Python 3.10+
- `CEREBRAS_API_KEY` or `OPENAI_API_KEY`
- `EXA_API_KEY`
