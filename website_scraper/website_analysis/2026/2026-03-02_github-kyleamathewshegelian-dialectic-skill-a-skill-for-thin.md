### TL;DR
A GitHub repository providing a complex "skill" for AI coding agents (like Claude Code or Cursor) that orchestrates a multi-agent Hegelian dialectic—using two opposing "Electric Monks" and an orchestrator—to automate deep reasoning, overcome cognitive biases, and synthesize complex semi-lattice structures from human problems.

### Key Quote
"This isn't artificial intelligence — it's an artificial belief system that frees you to think."

### Summary
**Tool/Library**: A prompt engineering "skill" designed to run within coding agents capable of spawning subagents and web search.

*   **Core Mechanism**: Utilizes three AI subagents:
    *   **Two Electric Monks**: Each adopts a fully committed, non-hedging position (Thesis/Antithesis) on a given topic.
    *   **Orchestrator**: Decomposes arguments, performs structural comparisons, and synthesizes the results.
*   **Theoretical Framework**: Combines Venkatesh Rao’s "Electric Monks" (belief offloading), John Boyd’s "Destruction and Creation" (decomposition/synthesis), Elizabeth Eisenstein’s "Typographic Fixity" (side-by-side comparison), and Christopher Alexander’s "Semi-lattice" theory (overlapping vs. tree structures).
*   **Process Workflow**:
    1.  **Elenctic Interview**: Socratic questioning to surface hidden assumptions.
    2.  **Monk Essays**: Agents write isolated, committed position papers.
    3.  **Determinate Negation**: Analysis of self-sublation and shared assumptions.
    4.  **Boydian Decomposition**: Shattering arguments into atomic parts to find cross-domain connections.
    5.  **Synthesis (Sublation)**: Creating a new concept that cancels and preserves previous insights (not a compromise).
    6.  **Hostile Audit**: A fresh agent attacks the synthesis for flaws.
    7.  **Recursion**: The synthesis becomes the thesis for the next round.
*   **Installation & Usage**:
    *   **Prerequisites**: Requires a coding agent (Claude Code, Cursor, Windsurf) that supports subagent spawning and web search.
    *   **Command**: `/install-skill /path/to/SKILL.md` followed by a topic description.
    *   **Cost**: Heavy process; requires 10–15 minutes per round and 3+ rounds for best results.
*   **Output Artifacts**: Generates a directory of Markdown files including `context_briefing.md`, `monk_a_output.md`, `monk_b_output.md`, `determinate_negation.md`, `sublation.md`, and `dialectic_queue.md`.
