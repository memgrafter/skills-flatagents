### TL;DR
OrbStack is free for personal use with a business tier at $8/user/month for commercial use, offering additional enterprise features like SAML SSO and priority support.

### Key Quote
"Always free for personal use"

### Summary
**Personal Tier (Free)**
- Personal, non-commercial use only
- Run containers
- Run Linux machines
- Community support

**Business Tier ($8/user/month)**
- $96 billed annually
- Commercial use licensed
- Debug Shell for container debugging
- Priority support
- Batch organization purchases
- SAML SSO
- Invoice & PO payments
- "More coming soon"
