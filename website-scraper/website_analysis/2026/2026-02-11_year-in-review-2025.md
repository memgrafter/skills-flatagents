### TL;DR
Mario Zechner's 2025 year-in-review documenting his deep dive into agentic coding workflows, building multiple AI tools (Yakety, VibeTunnel, Sitegeist, pi), investigating Austrian media's undisclosed LLM usage, and maintaining long-running hacktivist projects.

### Key Quote
"Nobody knows yet how to do this properly. We are all just throwing shit at the wall, declaring victory, while secretly crying over all the tech debt we introduced into our codebases by letting agents run amok."

### Summary

**Personal Context**
- Health setback in February/March (hospitalization, recovery) caused a gap in GitHub contributions
- Author is 41 years old, based in Austria

**Major Projects**

**Boxie** — Always-offline audio player for his child
- Self-designed PCBs, SMT soldering, 72 cartridges with audio plays/books/music
- Ported Doom to the embedded system
- Child still uses daily; hope it sparks technical curiosity

**Cards for Ukraine** — Charity with partner Tanja
- Crossed €300,000 in donations in 2025
- Sent 498 cards total (190 in May, 308 in October) to Ukrainian families in Austria
- Donations harder to come by after 3 years of war and recession

**heisse-preise.io** — Austrian grocery price scraper
- SPAR API change caused data gap (Dec 24, 2024 – May 14, 2025)
- Advocating for legalization of grocery price scraping and EAN disclosure
- Cory Doctorow referenced him as "the kid in Austria"

**AI/Agentic Coding Journey**

**Texty** (Jan 2025) — First "vibe coded" project
- Browser extension + Android app for dyslexia-friendly spell/grammar checking
- Used Accessibility Service on Android; abandoned due to clumsiness

**Claude Code Investigations**
- Patched binary to disable anti-debugging, enabled `/cost` for Max users
- Findings: Haiku generates wait messages, checks for injection attacks, system prompt caused file deletion
- Built tools: claude-trace, claude-bridge, cc-antidebug, cccost, claude-notify
- Created cchistory.mariozechner.at for tracking system prompt diffs

**MCP Abandonment**
- Built vs-claude (VS Code MCP server), mailcp (Gmail MCP)
- Concluded MCP outputs flood context; CLI tools are easier to build, debug, and compose
- Replaced with gmcli, gccli, gdcli (Google CLI tools)

**Yakety** — Local voice transcription app
- Holds hotkey to record, releases to paste transcription
- Uses local Whisper models; open sourced
- Still uses for writing, not coding

**VibeTunnel** — 24-hour hackathon project with Peter and Armin
- Control Claude Code from mobile phone
- "Frankenstein that worked but was basically unmaintainable"
- Peter later built Clawdis as a better version

**Sitegeist** — Browser extension for deep research and scraping
- Multi-provider/multi-model (Claude performs best)
- Creates artifacts (HTML, Excel, PowerPoint, Word)
- Skills system for automation; uses debugger for paywall bypassing
- Security concerns about prompt injection and credential exfiltration
- Plan to open source after cleanup

**Cellgeist** — Same concept for Excel (proof of concept)

**pi coding agent** — His own minimal coding agent harness
- No MCP, no sub-agents, no plan mode, no to-do tool
- Core tools: read, write, edit, bash
- Multi-provider/multi-model, cost tracking, hooks, custom UI contributions
- Open source with third-party contributions
- Exclusive daily driver for past 2 months

**Teaching Non-Technical Users**
- Taught linguist wife Claude Code in 2 sessions
- She automated 18,000+ row linguistic data analysis with reproducible Python scripts
- Key insight: agents effective for domain experts who can verify outputs
- Videos in German (4 parts), code on GitHub

**Investigations: Slop in the Wild**

**Austrian Media LLM Usage**
- Method: tracked em-dash/en-dash frequency normalized by article length
- OE24: stable 2021-2024, massive spike 2025 in Shopping, Tierschutz, lifestyle
- Differential analysis: if dashes up + hyphens down = CMS; if dashes up + hyphens flat = LLM
- Falter/Krone showed CMS pattern; Exxpress/OE24/Heute showed LLM pattern
- Taxpayer-funded: OE24 parent got €287K, Heute got €140K for "AI journalism"
- No disclosure to readers

**fobiss School AI Tools**
- Homework grading produces essentially random grades
- Sophie Scholl chatbot said "no injustice happened to me personally"
- Training videos use AI voiceover, minimal warnings about verification
- 114 Austrian schools in pilot project; final report is "glorified link collection"

**Facial Recognition Hiring Paper**
- The Economist covered pseudoscientific paper claiming facial analysis predicts personality
- Methodological garbage: unreliable data sources, weak R², laughable robustness checks
- "Phrenology with a fresh coat of AI paint"

**Other Investigations**
- Government chatbots: Kremsi jailbreakable, claims EU hosting but uses Vercel US
- Clinara medical transcription: 5-10% word error rate, exposed system prompt
- StoryOne book generator: produces hallucinated LinkedIn slop, FAZ wrote puff piece

**Other Notable Projects**
- Parliament Watch: tracks MP absences (2 hours to build)
- Carinthian Honors Dashboard: scraped 3 years of honors, classified gender
- Coalition Protocol Leaks: hosted leaked FPÖ/ÖVP and ÖVP/SPÖ/NEOS negotiation docs
- WaveMind: audio news transcription search (Falter Radio, Ö1, ORF), closed access
- Cremer Archive: scraped Der Standard photo blog

**Talks & Workshops**
- Vienna Tech Soirée panel with Armin Ronacher
- Claude Code Anonymous meetup (Vienna)
- LLMs in Real Software meetup (Graz)
- GrazJS meetup on JailJS and Sitegeist
- Bit & Babel livestream with Jenia
- Generative AI workshop for normies (2.5hr German livestream)
- Workshop at Profil (Austrian investigative weekly)
- Agentic coding workshop for local software company

**Conclusions**
- Biggest learning: enjoys meeting peers in real life
- Productivity increase unclear; projects have more technical depth
- Theme for the year: "It's all just vibes"
